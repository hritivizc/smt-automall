# SMT Automall Quick Links

Static landing page for SMT Automall with:

- Discount access page
- `docs/` copy ready for GitHub Pages

## Local preview

```bash
python3 -m http.server 4173
```

Then open `http://127.0.0.1:4173/`.

## GitHub Pages

Use the `docs/` folder as the Pages source on the `main` branch.
